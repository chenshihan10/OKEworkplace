## 一、项目概述

一个面向 OKX 交易所的**合约交易辅助分析系统**，提供：
- 实时价格监控（支持多币种添加/删除）
- 多维度技术分析（趋势、资金流、订单流、风险评分）
- 双向交易信号（做多/做空/观望/回避）
- 资金行为分析（盘整、吸筹、派发等阶段判断）
- 独立桌面窗口展示（不依赖浏览器）

## 六、待解决问题 / 可优化方向

### 优先级高

1. **OKX API 数据偶尔超时** — 3 个币种时 `refresh_all()` 偶尔耗时 6-8 秒（当前 5s 间隔 + `misfire_grace_time=10s` 已缓解），如果用户添加 10+ 个币种，数据新鲜度会明显下降。**建议**：单独 tikcer 轮询（高频）+ 其他指标按需/批量拉取。

2. **信号引擎尚未实盘验证** — `market_score.py` 的评分参数（权重、阈值）是基于经验预设的，需要在实际行情中验证和调参。

### 优先级中

3. **pywebview 窗口回退** — 如果用户系统没有 WebView2 Runtime，pywebview 会静默失败并回退到浏览器（已实现），但回退后不能联动关闭。**已在 Windows 10/11 自带的 Edge WebView2 上通过测试**。

4. **前端代码无构建/打包流程** — 前端是裸 HTML + JS，无模块化、无压缩。后续需要修改时可直接编辑 `index_v2.html` 和 `main.js`。

### 优先级低

5. **无量化回测模块** — 当前只有实时分析，不支持回测策略。

6. **无 PostgreSQL/数据库** — 所有数据在内存中，不落盘。服务重启后历史信号丢失。

7. **日志不完善** — 部分异常被 `logging.disable()` 或 try-except 静默处理，排查问题较困难。

## 九、注意事项

1. **OKX API 有频限** — 约 20 req/s，当前 3 个币种 × 4 个 API/ticker/candles/trades/funding = 12 次请求/轮询周期，在 5s 间隔下约 2.4 req/s，安全。添加更多币种请注意。
2. **EXE 打包路径固定** — `OKEworkplace.spec` 中的 `pathex` 指向 `d:\OKEworkplace\backend`，**新环境需要绝对路径一致**，否则打包失败。
3. **`.env` 文件对 EXE 是必需的** — 放在 `dist/OKEworkplace/` 中与 EXE 同目录。
4. **watchlist.json 自动生成** — 在 `backend/dist/OKEworkplace/` 下（EXE 模式）或 `backend/app/data/` 下（dev 模式），有自动迁移逻辑。

### 10.1 代理端口问题：主流解决方案 ✅ 已完成

**当前问题**：程序需要访问 OKX API（境外服务），在国内网络环境下可能需要代理才能正常访问。

**主流方案**：系统代理优先 + 智能检测 + 多路径回退

```python
# 建议修改 backend/app/services/oke_client.py

def _request_with_proxy(self, path, params):
    """
    主流策略：系统代理 → 用户配置代理 → 直连
    这是国内主流客户端（如微信、飞书、各种浏览器）的通用做法
    """
    attempts = []
    
    # 1. 系统代理（环境变量）- 最优先，用户在Windows设置中配置
    if os.environ.get('HTTP_PROXY'):
        attempts.append({
            'proxies': {
                'http': os.environ['HTTP_PROXY'],
                'https': os.environ.get('HTTPS_PROXY', os.environ['HTTP_PROXY'])
            }
        })
    
    # 2. 注册表读取系统代理（Windows专属）
    proxy_info = _read_windows_proxy_settings()
    if proxy_info and proxy_info != os.environ.get('HTTP_PROXY'):
        attempts.append({'proxies': proxy_info})
    
    # 3. 用户手动配置的代理（应用内设置）
    if self.custom_proxy:
        attempts.append({'proxies': {'http': self.custom_proxy, 'https': self.custom_proxy}})
    
    # 4. 直连（最终兜底）
    attempts.append({})
    
    # 逐个尝试，成功即返回
    for config in attempts:
        try:
            response = requests.get(url, **config, timeout=8)
            return response.json()
        except Exception:
            continue
    
    raise ConnectionError("所有网络路径均失败，请检查网络或代理设置")


def _read_windows_proxy_settings():
    """读取Windows系统代理设置（主流做法）"""
    try:
        import winreg
        with winreg.OpenKey(winreg.HKEY_CURRENT_USER, 
                            r"Software\Microsoft\Windows\CurrentVersion\Internet Settings") as key:
            enable, _ = winreg.QueryValueEx(key, "ProxyEnable")
            if enable:
                proxy_server, _ = winreg.QueryValueEx(key, "ProxyServer")
                if proxy_server:
                    return {'http': f'http://{proxy_server}', 'https': f'https://{proxy_server}'}
    except Exception:
        pass
    return None
```

**端口冲突解决方案**：

```python
def find_available_port(start_port=8000, max_attempts=20):
    """
    自动端口递增策略 - 主流应用的标准做法
    参考：VS Code、JetBrains系列、各种IDE都使用此策略

    for port in range(start_port, start_port + max_attempts):
        with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
```

### 10.3 后端静默启动与前后端联动关闭 ✅ 已完成

**当前问题**：后端启动时会显示控制台窗口，关闭前端时需要手动关闭控制台。

**主流方案**：`--windowed` + 任务栏托盘 + 关闭确认对话框

**优化后的 entry_exe.py**：

```python
import sys
import threading
import time
import signal
import ctypes
import socket

HOST = "127.0.0.1"
PORT = 8000
BASE_URL = f"http://{HOST}:{PORT}"


def find_available_port(start_port=8000, max_attempts=20):
    """自动端口递增策略"""
    for port in range(start_port, start_port + max_attempts):
        with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
            if s.connect_ex(('127.0.0.1', port)) != 0:
                return port
    raise RuntimeError("无可用端口")


def _start_server(port):
    """在后台线程中启动 uvicorn（静默模式）"""
    import uvicorn
    from app.main import app
    uvicorn.run(
        app, 
        host=HOST, 
        port=port, 
        reload=False, 
        log_level="warning",
        access_log=False,
    )


def _confirm_exit():
    """
    关闭确认对话框 - 主流应用的标准交互
    参考：VS Code、微信、各种专业软件都有此功能
    """
    result = ctypes.windll.user32.MessageBoxW(
        0,
        "确定要退出 OKEworkplace 吗？\n\n[是] 直接退出程序\n[否] 最小化到托盘",
        "退出确认",
        0x00000003 | 0x00000040  # MB_YESNO | MB_ICONINFORMATION
    )
    return result == 6  # YES = 6


def _create_tray_icon():
    """
    创建任务栏托盘图标 - 主流后台应用的标准做法
    参考：微信、QQ、钉钉、各种网盘客户端
    """
    try:
        import pystray
        from PIL import Image
        
        def show_window(icon, item):
            pass
        
        def quit_app(icon, item):
            from app.shutdown import shutdown_event
            shutdown_event.set()
            icon.stop()
            sys.exit(0)
        
        icon_image = Image.open("程序图标.ico")
        
        menu = pystray.Menu(
            pystray.MenuItem("显示窗口", show_window),
            pystray.MenuItem("退出", quit_app)
        )
        
        icon = pystray.Icon("OKEworkplace", icon_image, "交易分析终端", menu)
        return icon
    except Exception:
        return None


def _open_gui_window(port):
    """用 pywebview 创建独立窗口"""
    try:
        import webview
        
        def on_closed():
            """前端窗口关闭时的回调"""
            if _confirm_exit():
                from app.shutdown import shutdown_event
                shutdown_event.set()
                time.sleep(1)
                sys.exit(0)
            else:
                # 最小化到托盘
                pass
        
        webview.create_window(
            title="OKEworkplace - 交易分析终端",
            url=f"http://{HOST}:{port}/index_v2.html",
            width=1280,
            height=800,
            min_size=(960, 600),
            resizable=True,
            text_select=True,
            confirm_close=True,
            on_closed=on_closed,
        )
        webview.start()
        return True
    except Exception as e:
        return False


if __name__ == "__main__":
    # 信号处理
    def signal_handler(signum, frame):
        from app.shutdown import shutdown_event
        shutdown_event.set()
        sys.exit(0)
    
    if sys.platform.startswith('win'):
        signal.signal(signal.SIGINT, signal_handler)
        signal.signal(signal.SIGTERM, signal_handler)
    
    # 自动选择可用端口
    port = find_available_port(8000)
    
    # 启动后台服务线程
    server_thread = threading.Thread(target=_start_server, args=(port,), daemon=True)
    server_thread.start()
    
    # 创建托盘图标
    tray_icon = _create_tray_icon()
    if tray_icon:
        tray_icon.run_detached()
    
    # 打开 GUI 窗口
    _open_gui_window(port)
```
